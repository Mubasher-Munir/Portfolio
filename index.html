About

Full-stack range, Angular-first.

I'm a Principal Engineer Consultant with eight years across the full stack — Angular, from version 8 through 18, paired with ASP.NET Core and C# on the backend. Angular is where I go deepest, but most of the projects I've owned were mine end to end: UI, API, and the SQL Server schema underneath.

That range has taken me through enterprise supply chain software (real-time shipment tracking and EDI integration at Systems Limited), AI-powered document automation and RPA platforms at OpenBots AI, and a healthcare eligibility system I built front to back — Angular UI and ASP.NET Core APIs — for patient records, insurance, and copay logic. Along the way I've led engineering teams across as many as four concurrent products, and set the standards — front-end and API — those teams still build on.

Outside the day job, an MPhil in Computer Science let me apply NLP and deep learning to a genuinely under-served problem: detecting empathy in Urdu-language text.
